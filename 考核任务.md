## HTML学习笔记
**HTML：网页的“骨架工程师”** 
**核心定位**
 
**HTML**（HyperText Markup Language，超文本标记语言）是构建网页的基础，负责定义页面的结构和内容，相当于给网页搭建“骨架”，不涉及任何美化或交互逻辑。所有可见的文字、图片、视频等内容，都需要通过HTML标签组织起来。
 
**基础语法规则**
 
### 1. 标签结构：
​- 绝大多数标签为成对出现，包含“开始标签”和“结束标签”，内容包裹在中间（如  <h1>标题内容</h1> ）。
​- 少数单标签无需结束标签，通常用于插入资源（如  <img>  插入图片、 <br>  换行、 <input>  输入框）。
​- 标签可嵌套，但不能交叉嵌套（正确： <div><p>段落</p></div> ）。
​
### 2. 属性用法：
​- 标签可以添加属性，属性写在开始标签内，格式为  属性名="属性值" （如  <a href="https://www.baidu.com">百度</a> ）。
​- 常用通用属性： class （类名，可重复，用于批量控制元素）、 id （唯一标识，用于单独控制元素）、 title （鼠标悬浮时显示的提示文字）。
​
### 3. 文档结构：
一个标准的HTML文档必须包含固定结构，确保浏览器正确解析。

**常用标签**
### 1. 布局类标签（划分页面区域）
 
-  <div> ：通用块级容器，独占一行，可自定义宽高，示例为  <div class="header">头部</div> 。
​
-  <header> ：页面头部（如导航栏），语义化标签，更易读，示例为  <header>导航内容</header> 。
​
-  <section> ：独立内容区块（如章节），语义化标签，区分不同板块，示例为  <section class="chapter">章节内容</section> 。
​
-  <footer> ：页面底部（如版权信息），语义化标签，示例为  <footer>©2025 版权所有</footer> 。
​
-  <nav> ：导航栏区域，语义化标签，适合放链接，示例为  <nav><a href="#">首页</a></nav> 。
 
### 2. 文本类标签（展示文字内容）
 
-  <h1>-<h6> ：标题（h1层级最高，h6最低），字体加粗，自带margin，示例为  <h2>二级标题</h2> 。
​
-  <p> ：段落，自带上下margin，自动换行，示例为  <p>这是一段文字，会自动换行。</p> 。
​
-  <strong> ：强调文本（加粗），语义化强调，比 <b> 更推荐，示例为  <strong>重要内容</strong> 。
​
-  <em> ：强调文本（斜体），语义化强调，比 <<i> 更推荐，示例为  <em>强调内容</em> 。
​
-  <br> ：强制换行，单标签，无内容，示例为  <p>第一行<br>第二行</p> 。
​
-  <hr> ：水平线，单标签，分隔内容，示例为  <hr> 。
 
### 3. 媒体与链接标签（插入资源）
 
-  <img> ：插入图片，核心属性为src（图片地址）、alt（替代文本，图片加载失败时显示），示例为  <img src="img/1.jpg" alt="风景图" width="300"> 。
​
-  <a> ：超链接，核心属性为href（目标地址）、target（打开方式，_blank为新窗口），示例为  <a href="https://www.baidu.com" target="_blank">去百度</a> 。
​
-  <video> ：插入视频，核心属性为src（视频地址）、controls（显示播放控件），示例为  <video src="video/1.mp4" controls width="500"> 。
 
### 4. 列表标签（组织多条内容）
 
-  <ul>+<li> ：无序列表（默认圆点标记），示例为  <ul><li>苹果</li><li>香蕉</li></ul> 。
​
-  <ol>+<li> ：有序列表（默认数字标记），示例为  <ol><li>第一步</li><li>第二步</li></ol> 。
​
-  <dl>+<dt>+<dd> ：定义列表（标题+描述），示例为  <dl><dt>HTML</dt><dd>网页骨架</dd></dl> 。

  
## CSS学习笔记
**CSS：网页的“造型师”**
 
**核心定位**
 
**CSS**（Cascading Style Sheets，层叠样式表）负责美化网页，控制元素的颜色、字体、布局、间距等样式，相当于给HTML“骨架”穿上“衣服”。
 
引入方式（优先级：行内 > 内部 > 外部）
 
### 1. 行内样式（不推荐，耦合度高）
 直接写在HTML标签的  style  属性中，仅作用于当前标签。
 
### 2. 内部样式（适合小型页面)
 写在  <head>  中的  <style>  标签内，作用于整个页面。
 
### 3. 外部样式（推荐，适合大型项目）
 单独创建  .css  文件（如  style.css ），通过  <link>  标签引入HTML，可复用性强。
 
 **核心知识点：选择器** 
**选择器**是CSS的核心，用于定位HTML中的元素并应用样式。
 
### 1. 基础选择器
 
- 标签选择器：语法为标签名，作用是选中所有该标签的元素，示例为  p { color: red; } （所有 <p> 变红色）。
​
- 类选择器：语法为.类名，作用是选中所有class为该值的元素，示例为  .box { width: 100px; } （所有 class="box" 的元素宽100px）。
​
- ID选择器：语法为#ID名，作用是选中唯一ID为该值的元素，示例为  #title { font-size: 20px; } （ id="title" 的元素字号20px）。
​
- 通配符选择器：语法为*，作用是选中页面所有元素，示例为  * { margin: 0; padding: 0; } （清除所有元素默认边距）。
 
### 2. 复合选择器
 
- 后代选择器：语法为父选择器 子选择器，作用是选中父元素内的所有子元素，示例为  .nav a { color: black; } （ .nav 内的所有 <a> 变黑色）。
​
- 子选择器：语法为父选择器 > 子选择器，作用是选中父元素的直接子元素，示例为  .nav > a { color: black; } （ .nav 的直接子元素 <a> 变黑色，孙子元素不算）。
​
- 并列选择器：语法为选择器1, 选择器2，作用是同时选中多个选择器的元素，示例为  h1, h2 { font-weight: normal; } （ <h1> 和 <h2> 都取消加粗）。
​
- 伪类选择器：语法为选择器:伪类，作用是选中元素的特定状态，示例为  a:hover { color: red; } （鼠标 hover 时 <a> 变红色）。
 
**常用样式属性详解**
 
### 1. 颜色与背景
 
-  color ：控制文字颜色，取值示例有 red （英文）、 #ff0000 （十六进制）、 rgb(255,0,0) （RGB）。
​
-  background-color ：控制背景色，取值方式同 color 。
​
-  background-image ：设置背景图片，取值示例为 url("img/bg.jpg") 。
​
-  background-repeat ：控制背景图片是否重复，取值示例有 no-repeat （不重复）、 repeat-x （水平重复）。
​
-  background-position ：控制背景图片位置，取值示例有 center （居中）、 left top （左上）。
 
### 2. 字体与文本
 
-  font-family ：设置字体（优先用系统自带），取值示例为 "微软雅黑", sans-serif （无微软雅黑则用默认无衬线字体）。
​
-  font-size ：设置字号，取值示例有 16px （像素，常用）、 1.2em （相对于父元素字号）。
​
-  font-weight ：设置字体粗细，取值示例有 normal （正常）、 bold （加粗）、 400 （正常）、 700 （加粗）。
​
-  line-height ：设置行高（控制文字间距），取值示例有 1.6 （相对于字号的1.6倍，常用）、 24px （固定值）。
​
-  text-align ：设置文本对齐方式，取值示例有 left （左对齐）、 center （居中）、 right （右对齐）。
​
-  text-decoration ：设置文本装饰（如下划线），取值示例有 none （取消装饰，常用於清除 <a> 的下划线）、 underline （下划线）。
​
-  text-indent ：设置首行缩进，取值示例为 2em （首行缩进2个字符）。
 
### 3. 布局与盒模型
 
每个HTML元素都可以看作一个“盒子”，由 content （内容）、 padding （内边距）、 border （边框）、 margin （外边距）组成。
 
-  width / height ：控制盒子内容区的宽/高，取值示例有 200px 、 50% （相对于父元素）。
​
-  padding ：控制内边距（内容到边框的距离），取值示例有 10px （四周都10px）、 10px 20px （上下10px，左右20px）。
​
-  margin ：控制外边距（盒子之间的距离），取值方式同 padding 。
​
-  border ：控制边框（可分开写 border-width / color / style ），取值示例为 1px solid #000 （1px黑色实线）。
​
-  border-radius ：控制圆角（常用於卡片），取值示例有 5px （四角圆角）、 50% （圆形，需宽高相等）。
​
-  box-shadow ：控制阴影（增强层次感），取值示例为 0 2px 6px rgba(0,0,0,0.1) （水平偏移0，垂直偏移2px，模糊6px，黑色半透明）。
 
### 4. 弹性布局（Flex，重点！）
 
Flex布局是最常用的自适应布局方式，只需给父容器添加  display: flex ，即可灵活控制子元素的排列。
 
- 父容器属性：
​
-  display: flex ：开启弹性布局，必须写在父容器上。
​
-  justify-content ：控制子元素水平对齐方式，取值示例有 center （居中）、 space-between （两端对齐，中间留白）、 flex-start （左对齐）。
​
-  align-items ：控制子元素垂直对齐方式，取值示例有 center （居中）、 flex-start （上对齐）、 flex-end （下对齐）。
​
-  flex-wrap ：控制子元素是否换行，取值示例有 wrap （换行，适配移动端）、 nowrap （不换行，默认）。
​
- 子元素属性：
​
-  flex: 1 ：子元素平分父容器剩余空间，多个子元素都加 flex: 1 则等宽

## JavaScript学习笔记
**JavaScript：网页的“魔术师”**
**核心定位**
 
**JavaScript**（简称JS）是赋予网页交互能力的脚本语言，相当于给网页“注入灵魂”，能响应用户操作（如点击、输入）、修改页面内容、动态加载资源等。
 
引入方式（优先级：行内 > 内部 > 外部）
 
### 1. 行内脚本（不推荐，可读性差）
直接写在HTML标签的事件属性中，如  onclick （点击事件）。
 
### 2. 内部脚本（适合小型交互）
写在  <script>  标签内，可放在  <head>  或  <body>  末尾（推荐放 <body> 末尾，避免JS执行时HTML未加载完成）。
 
### 3. 外部脚本（推荐，适合大型项目）
单独创建  .js  文件（如  script.js ），通过  <script src="script.js"></script>  引入HTML。
 
**基础语法**
 
### 1. 变量与数据类型
- 变量声明：用  let （可修改）或  const （不可修改，推荐优先用），替代旧的  var 。
​- 常用数据类型：字符串（ "abc" ）、数字（ 123 ）、布尔值（ true / false ）、对象（ {name: "张三"} ）、数组（ [1,2,3] ）。
 
 
### 2. 函数（封装可复用的代码）
- 函数声明：用  function  关键字，或箭头函数（简洁写法）。
 
 
### 3. 条件判断（根据条件执行不同代码）
- 常用  if-else  语句，适合多条件判断； switch  语句适合单变量多值判断。

## 页面编后感

### 一、项目构想与实现情况
1. 我的构想：
想做一个以《红楼梦》为核心的阅读分享页，包含“名著简介、经典章节、人物赏析、读者感悟”4个板块，既要展示文学内容，又要做简单的交互（比如平滑导航），让页面更有“阅读感”。
​
2. 实现程度：
✅ 完成了所有核心板块的排版（简介图文、章节卡片、人物横向滚动）；
✅ 实现了响应式布局（电脑/手机都能看）；
✅ 加了平滑滚动等简单交互；
❌ 原本想加“章节内容展开”功能，因为JS基础不足暂时没实现，因时间原因没能完成“读者感悟”这个板块
 
### 二、技术使用说明
 
1. HTML（搭建骨架）：
​
- 用 <nav>``<section>``<footer> 划分页面结构，让代码更清晰；
​
- 用 <div> +类名组织每个板块的内容（比如 intro   chapters ）；
​
- 给交互元素加了 id 和 class ，方便CSS和JS调用。
​
2. CSS（增添颜值）：
​
- 用“深棕+米白”的配色呼应《红楼梦》的古典氛围；
​
- 用 flex 和 grid 做自适应布局；
​
- 加了 box-shadow 和 hover 动效，让页面更有层次感。
​
3. JS（赋予变化）：
​
- 写了 scrollIntoView 实现“平滑跳转”（点击导航或Banner按钮能慢慢滑到对应板块）；
 
### 三、遇到的难题与解决方法
 
1. 最大难题：人物赏析区想做“横向滚动”，但一开始用 flex 会自动换行，没法横向滑动。
​
2. 解决过程：
​
- 先查了W3School的 overflow 属性，知道了 overflow-x: auto 可以让容器横向滚动；
​
- 给外层容器加 overflow-x: auto ，内层 character-list 用 min-width: fit-content 保证内容不压缩；
​
- 试了几次调整宽度和间距，终于实现了横向滚动的效果。

​
